Etiquetas de texto

En esta pagina pondré en práctica algunas etiquetas de texto.

  1. strong

    2. La etiqueta strong es usada para simbolizar texto importante.

  2. em

    3. La etiqueta em es usada para enfatizar texto.

  3. mark

    4. La etiqueta mark es usada para resaltar texto como si de un marcador se tratase.

  4. b

    5. La etiqueta b es usada para destacar texto sin importancia, visualmente similar al strong.

  5. i

    6. La etiqueta i es usada para texto con tono de voz distinto al resto, letra cursiva.

  6. u

    7. La etiqueta u es usada para escribir nombres propios o texto incorrecto intencional, elicoptero.

  7. s

    8. La etiqueta s es usada para fragmentos de texto inexacto que ya no es relevante, Placeholder para la etiqueta s.

  8. span

    9. La etiqueta span es usada para agrupar contenido de texto:

    1 + 1 = 2

    Es un ejemplo de suma, se encuentra dentro de una etiqueta span.

Hay otras etiquetas que no son tan usadas como las anteriores, usaré algunas a continuación:

V2f = V2o + 2ax

Fórmula del movimiento rectilineo uniformemente acelerado.
